HTML 学习教程（一）

内容概要

HTML（hyper text markdown language）被称为超文本标记语言，一种用来描述网页的语言，本篇教程主要对HTML的主要标签进行整理~

HTML基本标签

HTML结构标签

body之前

<!DOCTYPE>标签：文档类型声明标签(并不属于HTML语言)

作用就是告诉浏览器使用哪种HTML版本来显示网页

lang语言种类

定义当前文档显示的语言:en 英语，zh-CN中文，定义为中文的文档也可以写英文，定义为英文的文档也可以写中文

meta标签

标签必须位于标签内部 ① 在标签中，可以通过标签的charset属性规定HTML文档使用哪种字符集，避免乱码.charset常用的值GB2312、GBK、BIG5和UTF-8 ② 标签中的name属性 ③ 标签中的http-equiv属性 ③ 标签中的content属性 ④ 标签中的scheme属性

body中

标题标签 < h1 >-< h6 > 双标签

段落标签 < p > 双标签

换行标签 < br/ > 单标签

文本格式化标签

  • 加粗
  • 倾斜
  • 下划线
  • 删除线

    < div >标签和< span >标签

    < div >和< span >是没有语义的，它们是一个盒子，用来装内容完成页面布局
  • < div >标签 双标签

一行只能放一个< div >标签，大盒子

  • < span >标签 双标签
    一行可以放多个< span >标签，小盒子

    图像标签

  • 图像标签 < img src=”img.jpg” alt=”替换文本” title=”提示” width =”500”/ >
    src是图像标签的必备属性，指明了图像的src路径
    alt属性值是文本，图像的替换文字
    title属性值是文本，图像的提示文本
    width属性值是像素，图像宽度像素值
    height属性值是像素，图像高度像素值
    border属性值是像素，边框粗细
    注：属性之间不分先后顺序，标签与属性、属性与属性之间以空格分隔；属性采用键值对的格式，即key=”value”的格式

  • 路径
    该部分省略

    超链接标签

  • 链接语法格式 < a href=”跳转目标” target=”目标窗口的弹出方式”>文本或图像或音视频或表格 </ a>
    href是超链接标签的必备属性，指定链接目标的url地址
    target用于指定链接页面的打开方式，_self为默认值，_blank为在新页面中打开

  • 链接类型
    外部链接 ：href参数值为”外部某个网页地址”
    内部链接 ：href参数值为”内部页面地址”
    空链接 ：href参数值为#
    下载链接 ：href参数值为”.exe 或者 .zip等文件格式的文件地址”
    锚点链接 ：href属性值指定为”#名字”的形式并目标位置标签添加id属性为”名字”

    注释< !– – >和特殊字符

    表格标签

  • 表格的主要作用是用来展示数据

  • 表格的基本语法

  • 表头单元格
    < th >中间内容居中加粗显示< /th >，放在< tr>标签对内部，一般是位于表格的第一行或者第一列

  • 表格相关属性

  • 注：表格标签属性不常用，一般通过css进行设置，写在< table>标签中

    属性名属性值作用
    alignleft（默认）,center,right相对于周围元素的对齐方式
    border1或""（默认）是否有边框（默认没有）
    cellpadding像素值，默认1像素单元边缘与内容间的空白
    cellspacing像素值，默认2像素单元格之间的空白
    width像素值或者百分比表格宽度
    height像素值或者百分比表格高度

  • 表格结构标签
    有些表格较长，为了更好地表达表格语义，可以将表格分为表格头部 < thead>和表格主体 < tbody>两个部分

  • 合并单元格
    跨行合并：rowspan=”合并单元格数”，目标单元格在最上一个单元格
    跨列合并：colspan=”合并单元格数”，目标单元格在最左边一个单元格
    相当于是给目标单元格加一个”属性”吧

    列表标签

  • 表格的主要作用是实现简洁有序的布局

  • 无序列表

    带有自己的样式属性，但是一般在css中设置

  • 有序列表

  • 自定义列表
    “一个大哥带着几个小弟”

    < dl>中< dd>和< dt>标签的个数是没有限制的，而且< dt>和< dd>是兄弟关系而非包含关系

    表单标签

  • 使用表单的目的是为了收集用户信息。在HTML中，一个完整的表单由表单域、表单控件（也称为表单元素）和提示信息组成

  • 表单域：包含表单元素的区域,会把< form>< /form>中的表单元素信息提交给服务器

    属性属性值作用
    actionurl地址用于指定接收并处理表单数据的服务器程序的url地址
    methodget或者post指定表单数据的提交方式
    name名称用于指定表单的名称，以区分同一页面的多个表单域

  • 表单控件（表单元素）：允许用户在表单中选择或者输入的内容控件

  • 表单控件分类：
    ① input输入表单元素：单标签，< input type=”属性值”/>， type属性设置不同的属性值来指定不同的控件类型
    type属性分类：

    属性值描述
    text输入文本
    password输入密码
    radio单选按钮
    checkbox复选框
    submit提交按钮
    reset重置按钮
    button定义可点击按钮（通常和JS结合）
    file文件上传
    hidden定义隐藏的输入字段
    image定义图像形式的提交按钮
    除type属性之外的其他属性：
    属性属性值描述
    name用户自定义对表单控件进行区分，单选按钮必须拥有相同的name才能实现多选一，复选框也必须有相同的名字
    value用户自定义规定input元素的值（文本 框中可以看到初始效果）
    checkedchecked规定该input元素首次加载时应当被选中
    maxlength整数规定输入字段中字符的最大长度
    name和value是每个表单控件都应该有的元素，以备后台人员使用：name是表单控件的名字，要求单选按钮和复选框都必须有相同的name checked属性主要针对单选按钮和复选框，主要作用是一打开某页面就可以默认选中某元素 < label>标签本身不属于表单标签，但是常与表单元素搭配使用，为< input>元素定义标注（< label>标签用于绑定一个表单元素，当点击< label>标签内的文本时，浏览器就会自动将焦点（光标）转到或者选择对应的表单元素上，增加用户体验） ``` rows属性的属性值为"显示的行数",cols属性的属性值为"每行的字符数"（实际开发中使用css进行格式的设置）
